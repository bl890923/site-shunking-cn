解析网站建设、优化与排名容易犯的10个误区

2016-09-05 / 建站知识

  

网站在搜索引擎中排名时升时降，那是很正常的事。很多站长在对待这个事情上，在分析其原因上，很容易走入一些误区，一旦网站排名下降，就开始回忆这两天自已对网站做了什么改动，并坚定地认为最近的改动是网站排名下降的原因。如前两天网站更换了服务器，过了几天网站排名下降了，马上就认为更换服务器对网站排名有影响，而实际上原因不是因为您更换了服务器，而是因为新的服务器不稳定，导致网站访问速度下降，经常打不开等情况。所以，原因不在于更换服务器这个动作，而是在于新服务器的本身问题，如果新的服务稳定正常，就不会影响到网站的排名。站长们在网站建设、优化与排名的时候，一定要走出这些误区，否则可能会让你越做优化与排名就越迷茫。

 
1.国外空间与国内空间对网站排名有影响，放到国外服务器的网站比放在国内服务器的网站排名要差。
 
这种说法并不对，搜索引擎如果百度，对海外站点没有歧视。只是说，因为一些国家法律法规等因素，如果普通用户也访问不了网站，那像百度这样的搜索引擎可能也访问不了。
 
2.同一个IP上的网站中的一个网站被惩罚时，其它同一IP的网站将会受连累，也被惩罚。
 
这种可能性极小，同一IP上的网站在某段时间被百度惩罚的原因并不是因为是同一IP而相连被惩罚，而可能是因为放在了同一个服务器，而这个服务器很不稳定而导航都被降权的，如速度很慢、经常打不开等。
 
3.google支持nofollow，百度不支持。
 
至少从2010年开始，百度也是支持nofollow的。
 
4.站点开了CDN（内容分发网络），对百度排名有影响，网站权重会下降。
 
现在，很多大型网站都在用CDN，站点开了CDN不会对排名有影响。但是某些CDN服务商会导致网站在某些地区访问不稳定，可能会影响Baiduspider的抓取。所以，有影响不是因为你开了CDN，而是因为你的CDN服务商在技术上等因素而导致你的站点访问上不稳定。
 
5.使用site语法查询网站收录情况时，如果网站首页不排在第一页，说明网站被降权了。
 
这种说法是不正确的，site语法查询的检索结果，更多情况下类似随机的排序，除了说明这些网页已经被收录以外，不能说明任何问题。
 
6.百度网站体积显示最大是125K，大于125k的网页对百度收录与排名有影响。
 
页面大小和搜索引擎的抓取之间没有直接关系。只是说过大的网页会有抓取截断；而内容部分也是一样的，过大会有抓取截断。当然，抓取截断的上限，会远大于索引截断的上限。而不是说网页大于125K就会对网页收录与排名有影响。
 
7.标题越长越好，可以包括很多关键词。
 
搜索引擎对标题的长度没有限制。但是很多人喜欢往标题里塞很多关键词，以为那样就全面开花，但很可能适得其反。这个原理同在一个页面上设超链个数一样。分一杯羹的人越多，每个人所得的羹越少。
 
8.修改网站首页的 meta description 网站会受到惩罚？
 
很多站长这样觉得，大多是因为网站是新站，网站的主题与内容等都还不是想得成熟，所以在网站建设完成，在上线之初可能会经常变更一下标题与description，而恰好在变更标题与description时，网站排名出现了排名下降的。原因是，新站都在搜索引擎的一下观察期，新站的内容一般也变化得比较大，排名的波动是很正常的，只是恰好在您变更标题与description时出现了排名下降情况，这不是原因。过于频繁的修改，未必会及时的反馈在摘要中。当然，经常改动标题与description不利于用户体验，搜索引擎也不会欢迎，但不能说修改网站首页的 meta description 网站会受到惩罚，问题在于一个度上。
 
 
9.更换服务器对排名有影响。
 
不会，出现更换服务器后网站排名有影响的情况，但问题不在于你更换的服务器，而是因为可能你新的服务器不稳定，如果访问速度下降，经常打不开，硬件问题等。
 
10.如果做了百度推广之后，一旦不做推广网站就很难再有好的排名。或者是说百度推广是影响自然排名。
 
做不做网站推广，与网站的自然排名没有关系。至于不做推广后网站排名很低，甚至找不到，原因只能从您的网站质量上找问题，不做推广的网站，在网站建设之初就开始很注重网站的质量，因为这样才有好的排名。而做推广的网站，只要花多点钱就能排第一，所以不用考虑网站质量。而一旦您不推广，没有了特权，和别的网站一样站在以质量取胜的同一条起跑线上的，自然是很难与别人比了。
